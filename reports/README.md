README file for reports.

 - architectural reports are version fixed reports that give full readable info on the architecture.

&#x20;- those reports only change with version change and are part of git repro.

&#x20;- figures reports important info before and after training a model.

&#x20;- validation reports are dataset validation reports as per the seed and number of samples per class.

&#x20;- statistical reports provide information of the generated dataset on those parameters it was generated.
